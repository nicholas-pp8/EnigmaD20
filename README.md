<h1 align="center">Enigma D20 WhatsApp Bot</h1>

<p align="center">
  A powerful, multi-purpose WhatsApp bot built with Node.js and Baileys.
</p>

---

## 🌟 Features
* 🎮 **Interactive Games:** Tic-Tac-Toe and Word Scramble (Mega Edition).
* 🎵 **Media & Downloads:** Fetch and download songs and lyrics directly.
* 👑 **Advanced Owner Controls:** 
  * Auto-read messages & status
  * Auto-typing & always-online presence
  * Bypass View Once (`.vv`)
  * Remote message deletion
* 🔄 **Smart Auto-Update:** Built-in self-healing GitHub sync for seamless server updates directly via WhatsApp.

## 🛠️ Tech Stack
* **Engine:** Node.js
* **Library:** @whiskeysockets/baileys
* **Hosting:** Optimized for Linux/Katabump containers.

## 🚀 Setup & Deployment
1. **Clone the repository:**
```bash
   git clone [https://github.com/nicholas-pp8/EnigmaD20.git](https://github.com/nicholas-pp8/EnigmaD20.git)
   cd EnigmaD20
   ```
2. **Install dependencies:**
```bash
   npm install
   ```

## 🔑 Pairing Mode (How to Connect)
> **📝 Note: No Third-Party Pairing Websites Required!**
> Many users face security concerns or difficulties with external pairing websites. To keep things safe and simple, this bot features a direct, built-in pairing system.

**Please follow these detailed steps to pair your WhatsApp:**

* **Step 1:** Open the `index.js` file located in the main folder of the bot.
* **Step 2:** Scroll down a little bit inside the file. You will see an option named `PAIRING_NUMBER`.
* **Step 3:** Enter your WhatsApp number inside the quotes along with your country code (For example: `"916290371061"` without the '+' sign).
* **Step 4:** Save the file and start the bot on your server/panel.
* **Step 5:** Check your panel's console/terminal. The bot will automatically generate and display an 8-digit pairing code for you!
* **Step 6:** Open WhatsApp on your phone -> Go to **Linked Devices** -> Click **Link a Device** -> Choose **Link with phone number instead** -> Enter the 8-digit code!
